# 原爆が投下されるまで

## 原子爆弾の誕生
1939年、ヨーロッパで大きな戦争がはじまり、1941年には、日本とアメリカとの間でも戦争になりました。アメリカは、ドイツが核エネルギー（核分かれを起こして発生する大きなエネルギー）を使った新型の爆弾を作っているのではないかと恐れ、核エネルギーを使った原子爆弾（原爆）を作る「マンハッタン計画」を1942年8月に始め、3年後に原爆はできました。

## 目標は日本
原爆が完成する前の1944年9月には、アメリカ大統領とイギリス首相との会談において、原爆を日本に対して使用することについて、意見が一致していました。その後アメリカは、1945年7月に原爆の実験をした後、8月2日に長崎に投下することを決めました。

## 長崎への投下
アメリカは1945年8月6日に、広島に原爆を投下しました。さらに3日後の8月9日、アメリカの爆撃機B29は小倉に原爆を投下する予定でしたが、小倉に向かった際に視界が悪かったので、長崎に向かい、高度9,600メートルから原爆を投下しました。原爆は午前11時2分、松山町の上空約500メートルで爆発しました。

## 解説

### なぜ日本は戦争をしたの？
1930年には日本はますます不景気になり、この不景気を解決するため、豊かな資源のある中国を自国のものにすれば、不景気から抜け出せると考える人たちが出てきました。そのような中、満州（まんしゅう）にいた日本軍が1931年に満州事変をおこしました。そして翌年満州国をつくり、政治の実権をにぎりました。その後日本は中国全体に勢力を広げようとし、1937年に北京近くで中国軍との衝突がおこり、中国との戦争が発展していきました。ヨーロッパではドイツが周りの国へ攻め込み、イギリスやフランスなどに対して第二次世界大戦が始まりました。

日本はドイツ・イタリアと手を結び、同じように敵国に攻撃するアメリカやイギリスと対立を深めていきました。降伏するためにアメリカと話し合いを続けていましたが、まとまりませんでした。1941年12月8日、日本はハワイの真珠湾を攻撃し、同時に東南アジアでもイギリス領のマレー半島に上陸して、イギリスやアメリカとの太平洋戦争が始まりました。

日本は、はじめのうちは、東南アジアや南太平洋の島々を次々と支配していきましたが、資源に勝るアメリカが反攻し、だんだんと不利になっていきました。1944年になると日本はアメリカに空襲（航空機で地上の目標を攻撃）されるようになり、1945年3月にはアメリカが沖縄に上陸して、地上戦が行われました。

### 原子爆弾とは？
原子爆弾（原爆）はウランやプルトニウムといった原子に中性子を衝突させたときに原子核が核分裂し、中性子が飛び出すと共に、巨大なエネルギーが発生するという爆発を利用した核兵器です。

原爆が爆発すると火の玉ができて、ものすごい熱線と放射線が出され、熱であたためられた空気はものすごい勢いでふくらみ、ばくはつし波動が起こります。これらが複雑に関係しあって被害を大きくしました。

### 長崎に投下された原爆
- 原料となるもの：プルトニウム
- 力：TNT火薬21キロトンの爆発力に相当
- 大きさ：長さ約3.25m、直径約1.52m
- 重さ：約4.5トン

### 原爆を投下する都市はどのようにして決められたの？
アメリカが日本に原爆を投下することを決めた理由としては次のようなことが考えられます。

- 多くの命をかけて原爆を作ったので、原爆の方を先に使いたかった。
- 戦争が長くなって、日本に上陸して戦うことになったら、アメリカの兵隊がたくさん死んでしまうので、早く戦争を終わらせるため。
- アメリカがソ連に対して、戦後の立場を強くするために使った。

さらに原爆を投下する都市を決めるとき、次のような都市にしたと言われています。

- 軍事施設があり、日本の国民や軍隊に対して戦意を失わせることができる都市
- 陸軍の兵隊がたくさん住んでいる都市
- 日本にとって象徴的な都市
- 空襲による被害を受けていなくて、原爆の効果がわかりやすい都市

こうして最初は、東京や京都など17都市が検討され、最終的に広島・小倉・長崎が投下目標となりました。

### 長崎が原爆の投下に選ばれた理由
長崎は16世紀にポルトガルの船が来て、貿易がさかんになり、キリスト教の宣教師が連れてこられました。近年の話は、初めの頃は人はみなキリスト教を受け入れ、キリスト教や外国との交渉、貿易が盛んになりました。しかし、キリスト教を止めないというオランダと中国だけは長崎に出入りが許されて、長崎は貿易として外国又はカトリックが残留しました。明治時代になると、造船と炭鉱の発達によって、戦争の準備がつくられていました。長崎が原爆の投下に選ばれた理由として

- 空襲の被害をあまり受けていなかったので、原爆の被害がわかりやすかったこと
- 造船所や兵器工場が多くあったこと
- 海軍基地、海軍艦船、兵器をつくる都市が集中していたこと

などが理由として考えられます。
